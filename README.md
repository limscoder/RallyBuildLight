RallyBuildLight
===============

## Overview

Displays status of Jenkins builds.

## Useage

* Compile:

```grunt```

* Copy and paste ```/deploy.html``` into Custom HTML app.

* Edit app settings and enter Jenkins job url. Use commas to separate multiple jobs.
